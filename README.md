# CrackMe Solutions

> *Write-up dan dokumentasi penyelesaian berbagai challenge CrackMe untuk belajar Reverse Engineering.*

---

# Tentang Repository

Repository ini berisi kumpulan write-up dari berbagai challenge **CrackMe** yang saya kerjakan sebagai bagian dari proses pembelajaran **Reverse Engineering**.

Setiap write-up mendokumentasikan proses analisis mulai dari memahami executable, mengidentifikasi logika program, hingga menemukan solusi. Fokus utama repository ini adalah **proses analisis, pemahaman konsep, dan teknik Reverse Engineering**, bukan sekadar memperoleh jawaban akhir.

Repository ini dapat digunakan sebagai referensi belajar bagi siapa saja yang ingin memahami cara kerja program secara lebih mendalam melalui pendekatan Reverse Engineering.

---

# Tujuan

Tujuan dari repository ini adalah:

- Mempelajari dasar-dasar Reverse Engineering.
- Melatih kemampuan analisis program executable.
- Memahami alur kerja debugger dan disassembler.
- Mendokumentasikan proses penyelesaian challenge secara sistematis.
- Menjadi arsip pembelajaran yang dapat digunakan kembali di masa depan.
- Berbagi pengetahuan dan pengalaman kepada komunitas yang tertarik pada bidang Reverse Engineering.
